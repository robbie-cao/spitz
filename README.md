# Multi-Thread File Service

A multi-thread file service:
- download
- upload
- query
- management
  * add
  * remove
  * ...

## Setup

```shell
$ sudo apt-get install curl wget
$ sudo apt-get install libcurl4-openssl-dev
$ sudo apt-get install libev-dev
$ sudo apt-get install sqlite3 libsqlite3-dev
```
